# Exercicis d'Avaluació Continuada (Òptica / Física UB)

En aquest repositori hi ha alguns dels exercicis proposats per fer a casa durant
els cursos anteriors de l'Assignatura d'Òptica del Grau de Física de la 
Universitat de Barcelona.

Veureu que està separat per carpetes segons el tema i, per cada tema,
hi ha un seguit de carpetes indicant el semestre en el que s'ha proposat l'exercici.
